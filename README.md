## ⚙️ DB 접속 정보 가이드
- **Host**: `localhost` (또는 본인의 MySQL 서버 IP)
- **Port**: `3306`
- **User**: `본인의 계정` (예: root)
- **Password**: `본인의 비밀번호`
- **Database**: `cafe_kiosk`

> **보안 주의**: 실제 데이터베이스 접속 정보는 보안을 위해 `dbinfo.properties` 파일에 별도로 작성하여 사용하시기 바랍니다. 이 파일은 `.gitignore`에 의해 Git 추적에서 제외됩니다.
