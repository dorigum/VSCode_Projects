# ☕ 카페 키오스크 관리자(Admin) 기능 관리 명세서

본 문서는 카페 키오스크 프로젝트의 관리자 시스템에 적용된 MVC 패턴과 계층형 아키텍처, 그리고 구현된 핵심 관리자 기능들을 상세히 분석하고 기록합니다.

---

## 1. 전체 아키텍처 및 계층별 역할

본 시스템은 **Layered Architecture**를 기반으로 하며, 각 계층은 엄격하게 분리된 책임을 가집니다.

### 🏛️ 계층 구조 (Layered Architecture)
- **View Layer (`view/`)**: 사용자 인터페이스(UI) 및 입출력 전담.
  - `MenuView`: 관리자 전용 메인 메뉴 및 서브 메뉴(카테고리/메뉴/회원/주문/통계) 루프 제어.
  - `EndView`: 성공 결과 및 데이터 목록(메뉴/통계/주문 상세) 시각화 출력.
  - `FailView`: 에러 메시지 및 비즈니스 예외 상황 출력.
- **Controller Layer (`controller/`)**: 요청 제어 및 흐름 관리.
  - `AdminController`: 서비스 계층 호출 및 결과에 따른 뷰 전환 전담. 모든 예외를 포착하여 일관된 에러 UI 제공.
- **Service Layer (`service/`)**: 핵심 비즈니스 로직 및 정책 강제.
  - `AdminServiceImpl`: 통계 집계, 지능형 매핑, 보안 정책(1인 관리자 등) 강제 로직 포함.
- **Repository Layer (`repository/`)**: 데이터베이스 영속성 관리.
  - JDBC 및 SQL을 통한 데이터 CRUD 및 트랜잭션 제어.

## 2. 구현된 핵심 관리자 기능

### 📁 2.1. 스마트 메뉴 및 카테고리 관리
- **지능형 메뉴-옵션 자동 매핑 (고도화)**:
    - 신규 메뉴 등록 시 이름에 특정 키워드(**'라떼', '프라푸치노'**)가 포함되면, 시스템이 자동으로 **'사이즈'와 '휘핑유무'** 옵션 그룹을 검색하여 즉시 연결합니다.
    - 이를 통해 관리자의 반복적인 데이터 입력 수고를 덜고 운영 생산성을 극대화했습니다.
- **카테고리 기반 옵션 일괄 설정**:
    - 개별 메뉴가 아닌 카테고리(예: 커피) 단위로 옵션 그룹을 매핑하여, 해당 카테고리에 속한 모든 메뉴에 공통 옵션을 한 번에 적용할 수 있습니다.

### 👤 2.2. 고도화된 회원 관리 및 보안 시스템 (New)
- **1인 관리자 정책 (Singleton Admin)**:
    - 시스템 안정성을 위해 관리자(`ADMIN`) 권한을 가진 계정은 오직 **단 1명**만 존재하도록 제한합니다.
    - 새로운 회원을 관리자로 승격시키려 할 때 기존 관리자가 있으면 승인을 거부하는 검증 로직이 작동합니다.
- **관리자 계정 삭제 보호**:
    - 관리자 계정은 실수로 삭제되는 것을 방지하기 위해 **직접 삭제가 차단**되어 있습니다.
    - 삭제를 원할 경우 반드시 먼저 '일반 회원'으로 등급을 변경한 후에만 가능합니다.
- **포인트 수정 보호**:
    - 관리자 등급 본인의 포인트는 수정할 수 없도록 제한하여 데이터 무결성을 확보했습니다.

### 🎁 2.3. 투명한 포인트 관리 시스템 (Point History) (New)
- **수정 사유 필수 기록**:
    - 특정 회원에게 포인트를 지급하거나 차감할 때, 관리자는 반드시 **'수정 사유'**(예: "생일 축하 포인트🎁", "잘못된 적립 취소")를 입력해야 합니다.
- **이력 추적**:
    - 모든 포인트 변동 내역은 사유와 함께 `POINT_HISTORY` 테이블에 기록되어, 나중에 관리자와 회원 모두가 확인할 수 있습니다.

### 📊 2.4. 비즈니스 통계 분석 (BI)
- **기간별/시간대별 매출 분석**: 시작/종료일 설정을 통한 객단가 분석 및 피크타임 시각화(콘솔 막대 그래프).
- **VVIP 회원 분석**: 누적 결제액 기준 상위 기여 회원을 추출하여 마케팅 지표를 제공합니다.

### 📦 2.5. 직관적인 주문 및 취소 관리
- **상세 주문 조회**: 주문 상태를 아이콘(`[V]`, `[X]`)으로 표시하고, 주문자가 선택한 **세부 옵션 정보**까지 계층형으로 출력하여 가독성을 높였습니다.
- **포인트 자동 정합성 처리**: 주문 취소 시 사용 포인트 반환 및 적립 포인트 회수를 트랜잭션으로 원자적으로 처리합니다.
